    Experiencia Profesional

    Encriptador de texto

    El encriptador y desencriptador de texto es un proyecto desarrollado como entrega final del primer módulo del programa de formación de Alura Latam. Este proyecto utiliza JavaScript, HTML y CSS para implementar una aplicación interactiva y funcional. Su principal objetivo es permitir al usuario transformar mensajes de texto mediante un sencillo proceso de encriptación y desencriptación.
